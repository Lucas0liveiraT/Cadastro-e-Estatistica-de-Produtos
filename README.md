# Cadastro-e-Estat-stica-de-Produtos
Sistema desenvolvido em Java para cadastro de produtos via console. Permite registrar nome, preço e quantidade, realizando cálculos como média de preços, valor total do estoque, produto mais caro, mais barato e item com maior quantidade. Projeto focado na prática de lógica de programação, estruturas de repetição, condicionais e formatação de saída.
