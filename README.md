# 📦 Sistema de Estatística de Estoque em Java

## 📌 Descrição
Aplicação desenvolvida em Java para cadastro de produtos via console. 
O sistema permite registrar nome, preço e quantidade, realizando automaticamente cálculos como média de preços, valor total do estoque, identificação do produto mais caro, mais barato e item com maior quantidade disponível.

## 🎯 Objetivo
Projeto criado com foco no aprimoramento da lógica de programação e prática de:
- Estruturas de repetição (`for`)
- Estruturas condicionais (`if`)
- Variáveis acumuladoras
- Manipulação de entrada com `Scanner`
- Formatação de saída com `printf`

## 🛠 Tecnologias Utilizadas
- Java
- Aplicação Console
- Programação Estruturada

## 📊 Funcionalidades
- Cálculo da média dos preços
- Cálculo do valor total do estoque
- Identificação do produto mais caro
- Identificação do produto mais barato
- Identificação do produto com maior quantidade

## 🚀 Melhorias Futuras
- Implementação com Programação Orientada a Objetos
- Criação de menu interativo
- Persistência de dados em arquivo
